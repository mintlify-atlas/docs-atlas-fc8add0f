The Marble dashboard is your central hub for creating and managing all your content. Built with a modern, intuitive interface, it provides everything you need to publish articles, organize content, and manage your team.

## Key features

Marble's dashboard is designed to help you focus on what matters most: creating great content.

## Dashboard structure

The dashboard is organized into several main sections:

### Content management

<Steps>
  <Step title="Posts">
    Create, edit, and publish articles. View all posts with filtering by status (draft/published), category, and featured posts.
  </Step>
  <Step title="Categories">
    Organize posts into categories for better content structure and navigation.
  </Step>
  <Step title="Tags">
    Add tags to posts for flexible content organization and discovery.
  </Step>
  <Step title="Authors">
    Manage author profiles, including bios, images, and social links. Link authors to user accounts or create standalone profiles.
  </Step>
  <Step title="Media">
    Upload and manage all media files used across your content.
  </Step>
</Steps>

### Workspace settings

Configure your workspace preferences, manage team members, set up API keys, configure webhooks, and manage billing.

<Info>
  Each workspace is completely isolated, with its own content, team members, and settings.
</Info>

## Getting started

When you first access the dashboard, you'll see your workspace overview with:

- **Recent posts** - Quick access to your latest content
- **Quick actions** - Create new posts, upload media, or manage team
- **Usage metrics** - API requests, media storage, and webhook deliveries

### Creating your first post

<Steps>
  <Step title="Navigate to Posts">
    Click **Posts** in the sidebar to view all your content.
  </Step>
  <Step title="Create new post">
    Click **New Post** to open the editor.
  </Step>
  <Step title="Write your content">
    Use the rich text editor to write your post. Add images, videos, code blocks, and more.
  </Step>
  <Step title="Configure settings">
    Set your post's category, tags, authors, and SEO metadata in the sidebar.
  </Step>
  <Step title="Publish or save as draft">
    Choose to publish immediately or save as a draft for later.
  </Step>
</Steps>

## Workspace switching

If you belong to multiple workspaces, you can switch between them using the workspace selector in the sidebar. Each workspace maintains its own:

- Content (posts, categories, tags)
- Team members and permissions
- Media library
- API keys and webhooks
- Billing and subscription

<Note>
  You can create unlimited workspaces on the Pro plan. The Hobby plan is limited to one workspace.
</Note>

## Navigation

The sidebar provides quick access to all dashboard features:

- **Home** - Workspace overview and recent activity
- **Posts** - All your published and draft content
- **Categories** - Content categorization
- **Tags** - Flexible content tagging
- **Authors** - Author profiles and management
- **Media** - Media library and uploads
- **Settings** - Workspace configuration, team, API, webhooks, and billing
