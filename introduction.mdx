# Welcome to Marble

Marble is a headless CMS that provides a super simple way to publish articles, product updates, and changelogs to your site. Built as a modern monorepo with a powerful REST API, Marble gives you the flexibility to manage content in a beautiful dashboard and deliver it anywhere.

## Use cases

Marble is perfect for:

- **Blog content** - Publish articles and tutorials to your website or app
- **Product updates** - Keep users informed about new features and improvements
- **Changelogs** - Maintain a public or private changelog for your product
- **Documentation** - Manage help articles and knowledge base content
- **Marketing content** - Deliver dynamic content to landing pages and marketing sites

## Architecture

Marble is built as a Turborepo monorepo with three main applications:

- **API** (`apps/api`) - Hono REST API running on Cloudflare Workers for blazing-fast edge delivery
- **CMS** (`apps/cms`) - Next.js 16 dashboard with React 19 for content management
- **Web** (`apps/web`) - Astro marketing site

Shared packages provide common functionality:

- **Database** (`packages/db`) - Prisma ORM with PostgreSQL (Neon serverless)
- **Editor** (`packages/editor`) - Tiptap-based rich text editor
- **UI** (`packages/ui`) - shadcn/ui component library

## Technology stack

- **Package Manager**: pnpm (v10.19.0+)
- **Node.js**: v20+
- **Database**: PostgreSQL with Prisma ORM
- **Cache**: Upstash Redis for rate limiting and caching
- **Authentication**: Better Auth with Google & GitHub OAuth
- **File Storage**: Cloudflare R2
- **API Framework**: Hono (edge-first, lightweight)
- **Dashboard**: Next.js 16 with React 19

## API example

Fetch your posts from anywhere:

<CodeGroup>
```bash cURL
curl https://api.marblecms.com/v1/posts \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.marblecms.com/v1/posts', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const { posts, pagination } = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.marblecms.com/v1/posts',
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
)

data = response.json()
posts = data['posts']
```
</CodeGroup>
